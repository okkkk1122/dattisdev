# 🌐 Enterprise Website - Multi-language Platform

یک پلتفرم Enterprise کامل با قابلیت‌های پیشرفته شامل چت آنلاین، سیستم تیکتینگ، و پنل مدیریت کامل.

## ✨ ویژگی‌ها

- 🌍 سیستم چندزبانه بومی (فارسی، انگلیسی، عربی)
- 💬 چت آنلاین Real-time
- 🎫 سیستم تیکتینگ پیشرفته
- 👨‍💼 پنل مدیریت کامل
- 📱 طراحی Responsive
- 🎨 انیمیشن‌های جذاب
- 🔒 امنیت بالا
- ⚡ Performance بهینه

## 🚀 شروع سریع

### پیش‌نیازها
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL
- Redis

### نصب و راه‌اندازی

```bash
# کلون کردن پروژه
git clone <repository-url>
cd dattisdev

# راه‌اندازی با Docker
docker-compose up -d

# یا راه‌اندازی دستی
cd frontend && npm install && npm run dev
cd backend && npm install && npm run start:dev
```
